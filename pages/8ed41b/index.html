# python

# python简介

python是一种解释型语言:这意味着它没有编译

python是交互式语言:这意味可以在Python提示符>>>后直接执行代码

python是面向对象语言:这意味着python支持面向对象的风格或代码封装在对象的编程技术.

# 基础语法

输出 : print(“hello”)

# 标识符

在Python中标识符和JAVA相似,有字母、数字、下划线组成.但不能以数字开头.

python中标识符是区分大小写的

用下划线开头的标识符是有特殊意义的, 以单下划线_foo的代表不能直接访问的类属性,需要通过类提供的接口进行访问,不能用from *** import *而导入

以双下滑线开头的__foo代表类的私有成员,以双下划线开头和结尾的__foo__代表Python里特殊方法专用的标识,如__init__()代表类的构造函数.

Python可以同一行显示多条语句,方法是用分号;分开如:print("你好");print("卧槽");

# 保留字符

关键字都是小写字母

# 行和缩进

Python不适用大括号{}来控制类,函数以及其他逻辑判断. 使用缩进来表示.

缩进的空白数量是可变的,但是所有的代码块语句必须包含相同的缩进空白数量,这个必须严格执行

# 多行语句

python中换行一般是语句的结束符

但是使用斜杠  可以将一行的语句分为多行显示

如果语句中包含 [] , {} 或 () 括号就不需要使用多行连接符

# python引号

单引号 ’’ 双引号 "" 三引号’’’或""" 来表示字符串

三引号可以由多行组成,可以来写多行文本

# 注释

单行注释用 #

多行注释可以用三引号

# 空行

函数之间或类的方法之间用空行分割,表示一段时间新的代码的开始.类和函数入口之间也用一行空行分隔,以突出函数的入口的开始.

空行不是语法的一种,只是方便维护

# 等待用户输入

msg = input("请输入信息:")

会等待用户输入,回车结束 可以在提醒信息中使用\n实现文本的换行

# 同一行显示多条语句

如果想在同一行中执行多条语句, 语句之间可以使用分号 ; 分割

# 输出

print()

# 多个语句构成代码块

缩进相同的一组语句构成uuyige代码块,称之为代码组

像if、while等等符合语句,首行以关键字开始, 以冒号: 结束, 改行之后的一行或者多行构成代码组.

首行及后面的代码成为一个子句

# python变量类型

变量存储在内存中的值, 这就意味着在创建变量时会在内存中开辟一个空间.

基于变量的数据类型, 解释器会分配指定内存,并决定什么数据可以被存储在内存中.

因此,变量可以指定不同的数据类型,这些变量可以存储整数,小数或字符

# 变量赋值

python中变量赋值不需要类型声明

每个变量在内存中创建,都包括变量的标识,名称和数据这些信息

每个变量在使用前必须赋值,变量赋值以后该变量才会被创建

等号= 用来赋值

python允许同时多个参数赋值, 如: a = b = c =1 或者 a,b,c = 1,2,jack

# 字符串

# 使用方法修改字符串的大小写

title()方法可以将首字母大写

msg = "hello world"print(msg.title())

# 合并(拼接)字符串

通过加好 + 拼接字符串

msg = "hello world" + "拼接"print(msg.title())
